# FileKeeper

FileKeeper is a personal project to help me manage file backups.

## Status

> **IMPORTANT:** This project is still in development.
>
> It is under active construction and is **not** feature-complete yet. Behavior, architecture, and APIs may change at any time.

## Tech Stack

- .NET 10
- Avalonia UI

## Target Platforms

- Linux
- Windows

## Project Goal

FileKeeper aims to provide a practical and simple way to organize, track, and maintain file backup snapshots.

## Notes

This is a personal project focused on learning, experimentation, and incremental improvements over time.
